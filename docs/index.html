Arquitetura de Software & CI/CD na Azure

Uma visão geral da arquitetura de microsserviços implantada na nuvem Azure, orquestrada com Kubernetes e totalmente automatizada com Terraform e GitHub Actions.

Diagrama de Fluxo e Ordem de Execução

Início do Processo

Este fluxo detalha o passo a passo executado do zero, para um melhor entendimento de como foi desenhado o fluxo com base nos repositórios.

Passo 1

TechChallenge-infra-foundational

Cria a base: Grupo de Recursos, VNet e o Estado Remoto do Terraform.

Passo 2 (Execução em Paralelo)

TechChallenge-infra-data

Provisiona os serviços de persistência: Azure SQL e Blob Storage.

TechChallenge-infra-compute

Provisiona as plataformas de computação: AKS, ACR e Function App.

Passo 3 (Execução em Paralelo)

TechChallenge

Compila e faz deploy do código da API principal para o AKS.

TechChallenge-app-function

Compila e faz deploy do código de autenticação para a Function App.

Passo 4

TechChallenge-infra-gateway

Cria o API Gateway e configura o roteamento para os serviços já implantados.

MVP Concluído

Arquitetura completa e funcional no Azure.
